### Project Overview

---
This Excel dashboard analyzes grocery sales performance across outlets, item types, and fat content. It highlights key metrics like total sales, average rating, and sales distribution by size, location, and outlet type to uncover top-performing categories and improvement areas.

### Exploratory Data Analysis

EDA was done to uncover insights and answer to key questions such as:

1. Which outlet size generates the highest total sales?
   
2. How do sales vary across different outlet locations (Tier 1, Tier 2, Tier 3)?
   
3. Which item types contribute most to total sales?
  
4. How does fat content (Low Fat vs Regular) affect sales performance?
  
5. Which outlet type records the highest average sales?

### Results

Here are the **results (insights)** for the questions based on the dashboard:

1. **Outlet Size:** Medium outlets generate the highest sales with **$507.9K (42%)**.
 
2. **Outlet Location:** **Tier 3 outlets** lead in sales performance.
 
3. **Item Type:** **Fruits and Vegetables** contribute the most to total sales.
 
4. **Fat Content:** **Low fat items** account for the majority of sales (**$776.3K, 65%**).
  
5. **Outlet Type:** **Supermarket Type 2** records the highest avg. sales (**$142**).
